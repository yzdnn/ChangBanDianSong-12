**～项目概述～**
  
「长坂点宋 12」是基于 「[KHドットフォントシリーズ](http://jikasei.me/font/kh-dotfont/)」中的「KHドット道玄坂12」补足简体字和部分繁体字的 12px 点阵粗宋体，适合在复古风格的游戏和招牌中使用。

**～缺字的救急对策～**

Q、缺字了！而且我现在十万火急立刻马上就要用这几个字哪怕我自己补都行求求你教教我怎么做

A、请看秘籍（？）↓

《连⑨也能看懂的⑨步补字教程》 by yzdnn

1、先安装[Java JDK](https://www.oracle.com/cn/java/technologies/downloads/#jdk25-windows)（安装时一直点“下一步”即可）、再下载字体编辑器[Bits'N'Picas](https://github.com/kreativekorp/bitsnpicas/releases/download/v2.1.1/BitsNPicas.jar)

2、下载本仓库Sources文件夹里的.bdf文件

3、打开Bits'N'Picas（询问打开方式就选Java(TM) Platform SE binary），选择File->Open...，打开你下载好的.bdf文件

4、在左边一栏选中Unicode(Full)，再单击右边的任意一个字符选中它

5、用输入法打出你需要补的那个字（如果输入的是词语，软件会自动跳转到最后一个字）

6、选中一个已有的汉字，Ctrl+C复制它，再选中你需要补的那个字，Ctrl+V粘贴

7、双击你粘贴好的字来打开它，再用左上角的铅笔工具和橡皮工具，把它画成你需要的样子

8、重复第5～7步，直到需要的字全部补完

⑨、Ctrl+Shift+E，确认导出格式为.ttf后点击Export，选择导出路径后，给字体起个新的文件名，确认，搞定☆

这只是一个临时救急对策，如果时间允许，或者自己导出来没法用，还是在[这里](https://github.com/yzdnn/ChangBanDianSong-12/issues/1)反馈缺字，等我来补吧_(:3」∠)_
